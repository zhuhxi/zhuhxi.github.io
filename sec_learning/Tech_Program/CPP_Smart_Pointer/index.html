Motivation

    此时就可以使用一种名为 Resource Acquisition Is Inialization (RAII)cpp_raii 的编程技术来解决这个问题。简单来说，就是把需要申请后才能使用的资源 (e.g. 堆内存的分配、线程、socket、文件描述符、互斥锁、磁盘空间和数据库连接等) 与对象 (object) 的生命周期相绑定，借助 C++ 运行时 (runtime) 对对象的生命周期的管理，来实现对这些资源的生命周期管理。auto_ptr 就是借助这种思想，其会利用析构函数来自动完成对所管理资源的释放，实例代码如下所示:
